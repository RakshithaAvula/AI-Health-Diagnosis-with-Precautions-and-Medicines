# 🏥 AI Health Diagnosis with Precautions and Medicines

## 📌 Project Overview
This project is a Machine Learning-based healthcare web application that predicts diseases based on user symptoms and provides:

- Disease description  
- Precautions  
- Medicine recommendations  

It uses Machine Learning algorithms (Decision Tree and Support Vector Machine) integrated with a Flask backend for real-time prediction.

---

## 🚀 Features
- 🧠 Disease prediction using symptoms  
- 💊 Medicine recommendations  
- 🛡️ Precaution suggestions  
- 📖 Disease descriptions  
- 🌐 User-friendly web interface  
- ⚡ Real-time prediction system

## 🏗️ System Architecture
User Input (Symptoms)
->
Data Preprocessing
->
Trained ML Model
(Decision Tree / SVM)
->
Disease Prediction
->
Precautions & Medicines
->
Output Display

---
## ⚙️ Working Process
1. User enters symptoms in the web interface  
2. Symptoms are converted into numerical format  
3. ML model processes symptom patterns  
4. Disease is predicted  
5. Precautions and medicines are retrieved from dataset  
6. Results are displayed to the user  

---

## 📂 Dataset Used
The dataset contains:
- Symptoms  
- Diseases  
- Precautions  
- Medicines  
- Disease descriptions
